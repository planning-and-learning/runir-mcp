# Dictionary: module-program `features.*`

Used by: [`../runir.ps.ext.counterexamples.md`](../runir.ps.ext.counterexamples.md).

| Column | Meaning |
|---|---|
| `id` | Feature alias, rendered as `fK`. |
| `symbol` | Module-program or per-module feature symbol. |

Rows are ordered. This order defines the `f0`, `f1`, ... feature columns in module-program `[states]` tables and the feature aliases used in `deltas` cells.
